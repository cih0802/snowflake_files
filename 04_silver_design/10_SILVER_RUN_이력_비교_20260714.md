## 🔴 회귀 원인 분석 & 수정 (2026-07-14)

### 원인
- BRONZE_GA4 샤드 테이블이 **소문자 인용 식별자** `"events_20260501"`로 저장됨(287,025행 존재).
- `macros/ga4_union_shards.sql`의 샤드 탐색 필터가 **대소문자 구분** 방식이었음:
  - `table_name LIKE 'EVENTS_%'` → 소문자 `events_...`와 매칭 **0건**
  - `REPLACE(table_name,'EVENTS_','')` → 소문자라 치환 실패 → BETWEEN 필터도 탈락
- 결과: 매크로가 샤드 0개로 판단 → `WHERE 1=0` 빈 결과 반환 → GA4 5모델 0행 → 하류 IDENTITY_MEMBER_XREF도 0행.
- run 은 SQL 오류 없이 SUCCESS(빈 결과도 유효 SQL) → **조용한 데이터 회귀**.

### 진단 근거 (실측)
| 검증 | 결과 |
|---|---|
| `COUNT(*) FROM BRONZE_GA4."events_20260501"` | 287,025 (원천 정상) |
| 기존 필터 `table_name LIKE 'EVENTS_%'` 매칭 수 | **0** |
| 수정 필터 `UPPER(table_name) LIKE 'EVENTS_%'` 매칭 수 | 1 |
| `REPLACE(UPPER(...),'EVENTS_','')` | 20260501 (날짜 정상 추출) |

### 수정 내용 (`macros/ga4_union_shards.sql`)
1. 샤드 탐색: `UPPER(table_name) LIKE 'EVENTS\_%' ESCAPE '\'` + `REPLACE(UPPER(table_name),'EVENTS_','')` (대소문자 무관).
2. FROM절 테이블 참조: `BRONZE_GA4."{{ t }}"` (소문자 인용 식별자 보존).

## 비교 판정 기준

| 판정 | 조건 | 해석 |
|---|---|---|
| ✅ 정상(멱등) | run 후 ROW_COUNT ≈ run 전 (원천 증분 범위 내) | INSERT OVERWRITE 정상 재생성 |
| ⚠️ 증가 | run 후 > run 전 | 원천 신규 입고분 반영(정상) 또는 중복 삽입(점검) |
| 🔴 감소/0 | run 후 < run 전 (ERP_BIZ_TARGET 제외) | 원천 누락·필터 오류·조인 실패 — 원인 조사 필요 |
| 🔴 LAST_ALTERED 미갱신 | run 후 LAST_ALTERED 가 run 전과 동일 | 해당 모델 미실행(select 범위 확인) |

### run 후 자동 비교용 쿼리 (스냅샷 테이블을 만들 경우)

```sql
-- (선택) run 전 스냅샷을 테이블로 보존해두면 run 후 JOIN 비교 가능
CREATE OR REPLACE TABLE GN_DW.SILVER._SNAPSHOT_BEFORE_RUN_20260714 AS
SELECT TABLE_NAME, ROW_COUNT, BYTES, CREATED, LAST_ALTERED
FROM GN_DW.INFORMATION_SCHEMA.TABLES
WHERE TABLE_SCHEMA = 'SILVER' AND TABLE_TYPE = 'BASE TABLE';

-- run 후 비교
SELECT b.TABLE_NAME,
       b.ROW_COUNT AS BEFORE_ROWS,
       a.ROW_COUNT AS AFTER_ROWS,
       a.ROW_COUNT - b.ROW_COUNT AS DELTA_ROWS,
       CASE WHEN a.LAST_ALTERED > b.LAST_ALTERED THEN 'RE-RAN' ELSE 'UNCHANGED' END AS RAN_STATUS
FROM GN_DW.SILVER._SNAPSHOT_BEFORE_RUN_20260714 b
JOIN GN_DW.INFORMATION_SCHEMA.TABLES a
  ON a.TABLE_NAME = b.TABLE_NAME AND a.TABLE_SCHEMA = 'SILVER'
ORDER BY ABS(a.ROW_COUNT - b.ROW_COUNT) DESC;
```
