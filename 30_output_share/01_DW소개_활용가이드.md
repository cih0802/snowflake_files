## 2. 표준 지표 정의서 (KPI Glossary)

| 지표 | 산출 기준 | 위치 |
|---|---|---|
| **개발(명)** `DEV_MEMBERS` | 당월 신규 개발 회원 수 COUNT(DISTINCT) | `WIDE_MEMBER_MONTHLY` |
| **개발(건)** `DEV_CNT` | SUM(개발금액) / 10,000 | `WIDE_MEMBER_MONTHLY` |
| **활동회원** `ACTIVE_MEMBERS` | 월말 기준 정상 납입 또는 유효 결연 상태 | `WIDE_MEMBER_MONTHLY` |
| **이탈(건)** `CHURN_CNT` | SUM(취소 + 감액) / 10,000 *(표준 신규 정의)* | `WIDE_MEMBER_MONTHLY` |
| **증액(명)** `INCREASE_MEMBERS` | 당월 증액 회원 수 | `WIDE_MEMBER_MONTHLY` |
| **미납(건)** `UNPAID_CNT` | 당월 미납 발생 건수 | `WIDE_MEMBER_MONTHLY` |
| **납입회비(원)** `PAID_FEE` | 당월 실납입 총액 (회비 + 기부금) | `WIDE_MEMBER_MONTHLY` |
| **회원구분** `MEMBER_TYPE` | 정기후원(FDRM) / 일시후원(ONCE) 통합 기준 | `DIM_MEMBER` |
| **서신참여** `LETTER_PART_*` | 발송 후 서신 참여 회원수/건수 | `WIDE_SERVICE_EVENT` |
| **+5일차 반응** `D5_*` | 발송일+5일 내 증액·서신·선물금·중단 | `WIDE_SERVICE_EVENT` |

> **회원구분 표준화**: 정기/일시는 원천 시스템에서 별도 관리되나, DW에서 단일 기준으로 통합. 전 부서가 동일 수치를 참조합니다.

## 4. 비즈니스 도메인 관계도

```
                         ┌─────────────────┐
              ──가입──▶   │  캠페인 / 브랜드 │  ──귀속──▶  ┌────────────┐
              │          └─────────────────┘             │ 조직 / 부서 │
   ┌──────────────┐                                       └────────────┘
   │ 회원 / 후원자 │ ──참여──▶  [ 행사 · 서신 · 발송 ]
   └──────────────┘
              │
              ──결제──▶  [ 납입 · 청구 · 예산 ]
```

- **중심 축**: 회원/후원자 → 캠페인/브랜드 → 조직/부서
- **활동 축**: 회원/후원자 → 행사·서신·발송
- **재무 축**: 회원/후원자 → 납입·청구·예산
