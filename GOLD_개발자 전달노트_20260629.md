# GOLD 테이블 만들기 안내 (GN_DW)

작성: 조인환 프로 · 수신: 굿네이버스 IT팀 · 2026-06-29

Gold 스키마의 테이블 설계를 24개로 추출했습니다. **CSV 파일이 설계도**이며, 나머지 문서는 참고용입니다.

## 테이블 생성 위치
- 스키마: **`GN_DW.GOLD`**
- 테이블 24개 = 분류표(DIM) 15개 + 수치표(FACT) 9개

## 첨부파일 설명
>'gold 스키마 컬럼 인벤토리_20260629.xlsx'
한 행이 컬럼 하나입니다.

| 컬럼명 | 의미 |
|---|---|
| 테이블명 / 컬럼명 | 만들 표 이름 / 컬럼 이름 |
| 타입 | 데이터 종류 (예: `NUMBER`, `VARCHAR`, `DATE`) |
| NULLABLE | `N`=값 필수(NOT NULL), `Y`=빈값 허용 |
| 키 | `PK`=기본키, `FK→표`=다른 표를 가리킴 |
| FK_타깃 | 그 컬럼이 가리키는 표와 컬럼 |

- 모든 표에 `DW_SOURCE_SYSTEM`, `DW_LOAD_TS`, `DW_UPDATE_TS`, `DW_BATCH_ID` 4개가 공통으로 들어갑니다.

## 생성 규칙 (3개만)
1. **분류표(DIM)를 먼저, 수치표(FACT)를 나중에** 만들어주시길 바랍니다. (순서: `DIM_DATE` → `DIM_ORG` → `DIM_MEMBER` → 나머지 DIM → FACT)
2. **FK는 FK_타깃에 적힌 표·컬럼으로 연결**해주시길 바랍니다.
3. FK_타깃에 **`※비강제`라고 적힌 컬럼은 FK를 걸지 마십시오**. 그냥 일반 컬럼으로 만들고 연결은 나중에 하겠습니다.

## PENDING
- **타입 길이**(예: `VARCHAR` 글자 수)는 아직 미정 → 엑셀 정의대로 생성 후 변경 예정입니다.
- 엑셀에 **없는 비율·증감·누계 같은 값**은 테이블이 아니라 별도 계층에서 계산합니다.
- 설명에 **⚠️ 표시된 컬럼**은 확인 중입니다. 

## 참고
- ERP·광고·GA 데이터는 아직 안 들어왔습니다. **컬럼은 그대로 만들고**, 데이터는 나중에 채울 예정입니다.
