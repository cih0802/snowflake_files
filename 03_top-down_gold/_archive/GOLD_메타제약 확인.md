# GOLD 메타 / 제약 보존 (6단계 산출물)

> 입력: `GOLD_지표 분류.md`(시간·합의 태그) + `신규지표 정보.md`(비고 원문) + `공통지표 정보.md`(비고 검증).
> 목적: 215개 지표의 **시간가용성·우선순위·합의/미해결 사항**을 GOLD 레이어에 메타로 보존하고, 미해결 합의 항목을 단일 리스트로 추적한다.
> **원칙(추정 금지)**: 본 문서의 시간·우선순위·합의 값은 **전부 원본 비고에 근거한 값만** 기재한다.
> - 신규 53개: 비고 원문 **1:1 재대조 완료**(시간버킷 24종·"꼭 필요함" 16건 — 6.1 검증로그).
> - 공통 162개: 원본 비고는 **#51(월말활동회원 계산조건)·#147(정의문) 2건 외 전부 공란**(키워드+광역 2회 grep 검증). 비고에 **시간제약 표기는 0건** → 시간가용성 **전체가능**(작업계획 원칙8). #51은 시간제약이 아닌 **계산 규칙**이므로 4.1에 "보존 대상"으로 기재. derived 정의 결함(identity·회비모델 등)은 1단계 설계 분석으로 식별된 항목이며 "원본 비고 기반"과 구분 표기한다.

## 1. 메타 보존 규칙

GOLD는 star schema 물리테이블에 분석 메타를 직접 담기 어려우므로 **2계층**으로 보존한다.

### 1.1 시간가용성 — 컬럼 COMMENT + 카탈로그 메타테이블 병행
- **enum**(원본 비고에서 도출, 6종):
  `전체가능` · `24년~`(24년 이전 불가) · `24.2~`(24년 2월~) · `25년~` · `2개년`(최근 2개년 롤링) · `적용불가`(현재 기준 산출 불가).
- 물리 컬럼에는 측정값별 지표# COMMENT를 이미 보존(5단계 DDL). 시간가용성은 **컬럼 단위로 달라지지 않고 derived(SV metric) 단위**에 주로 걸리므로, COMMENT가 아닌 **메타테이블 `GN_DW.GOLD.META_METRIC`** 을 단일 진실원천(SSOT)으로 둔다(아래 5절).
- 운영 데이터 입고(6월말) 후 SV metric 정의 시 `TIME_AVAILABILITY`를 필터/주석으로 강제 → 24년 이전 구간 조회 시 NULL/제외 처리.

### 1.2 우선순위("꼭 필요함") — 메타 플래그
- 신규 비고에 **"꼭 필요함"** 명시 = **16건**(아래 3절). `META_METRIC.PRIORITY_FLAG='꼭필요함'`으로 보존.
- 공통 162개에는 "꼭 필요함" 표기 없음(원본 검증 완료) → PRIORITY_FLAG NULL.

### 1.3 합의/미해결 — 메타 + 미해결 리스트
- 합의상태 `AGREEMENT_STATUS` ∈ {`OK`, `합의필요`, `적용불가`}.
- 미해결 항목은 별도 테이블 `GN_DW.GOLD.META_OPEN_ISSUE`로 추적(아래 5.2). 각 항목에 **차단성격**(design/execution/value)을 부여 — "설계는 방어 완료, 값만 채움" 원칙 가시화.

### 4.3 현업 확인 항목 Q1~Q9 (grain 방어 완료 — 답이 무엇이든 구조 무손상)

## 5. 메타 테이블 DDL 제안 (5단계 DDL에 append 가능)

> 물리테이블 COMMENT만으로는 derived(SV) 81개·시간/합의 메타를 담지 못하므로 카탈로그 메타테이블을 별도 제안. **GOLD star schema 18테이블(DIM 12 + FACT 6)과 분리**되며, 채택 시 `GOLD_ddl 초안.sql` 말미에 추가.

```sql
-- 메타 카탈로그: 215개 지표 전수 (시간가용성·우선순위·합의·배속 SSOT)
CREATE TABLE IF NOT EXISTS GN_DW.GOLD.META_METRIC (
    METRIC_SRC        VARCHAR       NOT NULL COMMENT '공통 / 신규',

    SOURCE_SYS        VARCHAR       COMMENT 'CRM / GA4 / ERP / AGENCY / GADS / ADMIN / 복합 / 확인필요',

    AVAILABILITY_NOTE VARCHAR       COMMENT '시간제약 원본 비고',

    AGREEMENT_NOTE    VARCHAR       COMMENT '합의 필요 원본/분석 내용',

    TARGET_ASSET      VARCHAR       COMMENT '배속: FMM/FTG-D/FTG-B/FSE/FGA/FAD/DIM_*/SV',
    CONSTRAINT PK_META_METRIC PRIMARY KEY (METRIC_SRC, METRIC_NO)
) COMMENT='지표 메타 카탈로그(시간가용성·우선순위·합의 SSOT). GOLD star schema와 분리';
```
